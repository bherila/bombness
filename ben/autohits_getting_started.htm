Getting Started

 

Getting Started with AutoHits 2005 couldn't be easier! Our software makes it simple to add surf URLs and actively surf all of your autosurf programs simultaneously. And you don't see a thing when you don't want to. You can be up and running in just a few short minutes.

Adding AutoSurf URLs to AutoHits 2005

To add an AutoSurf URL, locate the Site Management Grid in the main window of AutoSurf 2005. It should be directly to the left of this help window. Click on the last item in the list and press the down-arrow key to create a new item. Place a site name in the first column, and then its surf URL in the second column. Repeat this process as necessary; there is no maximum number of items, but we recommend you use no more than 20 AutoSurf programs at once unless you have an unusually large amount of RAM on your machine.

Once you have added all of your AutoSurf URLs, click the Refresh button at the top of the window to make your changes take effect.

Importing/Exporting Settings to/from a File

AutoSurf 2005 makes it easy to import your AutoSurf URL settings. Just click the Import button in the AutoHits 2005 main window, and a dialog will prompt you to select the exported settings file. In the same way, you can export your settings. Click the Export button and a dialog will appear to allow you to save the file in any folder or network location.
